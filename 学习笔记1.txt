在某个适合的点创建一个文件夹
打开文件夹之后，在空白区域右击 后点击BUSHHERE 
打开git
接下来写命令行代码；
第一步：
$ git init -b main（初始化）

回车之后会出现
Initialized empty Git repository in C:/Users/86183/Desktop/xuexi/.git/

第二步：
$ git remote add origin "https://github.com/jinhoupu/ABC.git"（该地址为自己在GitHub上创建的仓库地址）{这一步按我自己的理解是连接远程仓库（即GitHub上的仓库）}
回车之后不会弹出任何东西；

第三步：
$ git pull 
回车之后会弹出一些东西（原本以为这些都是不必要的，结果全线崩溃）
$ git pull "https://github.com/jinhoupu/ABC.git" main
回车之后会弹出
From https://github.com/jinhoupu/ABC
 * branch            main       -> FETCH_HEAD

第四步：
素质三连：
git add .
git commit -m "填写你要表达的信息"
回车之后会出现以下信息：
[main e022dea] 实用教程
 1 file changed, 1 insertion(+)
 create mode 100644 xuexi.txt

git push origin main
回车之后会出现以下一些信息
Enumerating objects: 4, done.
Counting objects: 100% (4/4), done.
Delta compression using up to 8 threads
Compressing objects: 100% (3/3), done.
Writing objects: 100% (3/3), 331 bytes | 331.00 KiB/s, done.
Total 3 (delta 1), reused 0 (delta 0), pack-reused 0
remote: Resolving deltas: 100% (1/1), completed with 1 local object.
To https://github.com/jinhoupu/ABC.git
   21c0db5..e022dea  main -> main
